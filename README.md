# Construction Analytics Platform

> An end-to-end Business Intelligence portfolio project demonstrating database design, SQL development, data modeling, Power BI, Tableau, and construction analytics.

---

## Overview

The Construction Analytics Platform is a portfolio project that simulates the analytics environment of a commercial construction company.

The project focuses on designing a normalized MySQL database, developing reusable SQL reporting views, and creating interactive dashboards in Power BI and Tableau. The goal is to demonstrate technical and analytical skills while solving realistic construction business problems.

---

## Project Objectives

- Design a normalized relational database
- Develop reusable SQL reporting views
- Create executive and operational dashboards
- Design a reusable analytics platform that supports multiple business intelligence tools.
- Document the complete solution using professional software development practices

## Solution Architecture

```
            MySQL Database
                   │
                   ▼
        SQL Reporting Views
                   │
        ┌──────────┴──────────┐
        ▼                     ▼
    Power BI             Tableau
```

Business logic is centralized in SQL reporting views, allowing both visualization platforms to consume the same validated metrics and KPIs.
